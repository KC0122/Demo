*****************************
Set up instructions
***************************

-Before you clone the report, you successfully installed playwright
-You can reffer to Playwright's  documentations for more information
https://playwright.dev/docs/intro

-Then pull all changes from below the repo to your root folder.
  https://github.com/KC0122/Demo.git

-Once done, you are good to go!


*****************************
To execute the tests
***************************

To run tests -command line -Issue the below command
 npx playwright test
 
To run tests in UI -Issue the below command
 npx playwright test --ui
